# API Bancária

## Descrição

API RESTful para simular operações bancárias básicas, incluindo autenticação JWT, transferências (PIX e agendadas), consulta de saldo e extrato.

## Tecnologias

- Ruby on Rails
- PostgreSQL
- JWT
- Sidekiq
- Docker
